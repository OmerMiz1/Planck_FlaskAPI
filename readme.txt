Prerequisites:
    1. Make sure you have "Flask" module
        (use "python -m pip install Flask")
    2. make sure you have "lxml.html" module
    3. Set enviromental variables:
        Windows: "set FLASK_APP=app.py"
        Linux: "export FLASK_APP=app.py"

Running the service:
    In your terminal, type: "flask run"

Running the test:
    Simply run the "cli_tester.py" file located in the tests folder and follow
    the instructions.
